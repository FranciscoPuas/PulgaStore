# Pulga Store

_Evaluación final módulo 2 - Desarrollo Web_

## Descripción

Pulga Store es una tienda online de tecnología de vanguardia, desarrollada como proyecto final del módulo 2. Incluye catálogo de productos, carrito de compras interactivo, filtros, búsqueda, y formulario de contacto, todo con diseño responsive y experiencia de usuario moderna.

## Características

- Catálogo dinámico de productos destacados (smartphones, laptops, tablets, accesorios, gaming).
- Filtros por categoría y búsqueda en tiempo real.
- Carrito de compras con animaciones, edición de cantidades y resumen de pedido.
- Checkout simulado con SweetAlert2 y notificaciones con Toastr.
- Diseño responsive con Bootstrap 5 y estilos personalizados.
- Animaciones de entrada con AOS.
- Formulario de contacto funcional.
- Código modular y buenas prácticas de desarrollo.

## Tecnologías utilizadas

- HTML5, CSS3, JavaScript (ES6+)
- [Bootstrap 5](https://getbootstrap.com/)
- [jQuery](https://jquery.com/)
- [AOS](https://michalsnik.github.io/aos/)
- [Owl Carousel](https://owlcarousel2.github.io/OwlCarousel2/)
- [SweetAlert2](https://sweetalert2.github.io/)
- [Toastr](https://codeseven.github.io/toastr/)
- [Font Awesome](https://fontawesome.com/)

## Uso

1. Clona o descarga el repositorio.
2. Abre `index.html` en tu navegador para ver la tienda.
3. Navega por las categorías, agrega productos al carrito y prueba el formulario de contacto.
